BFN [Text] Cairo, 4 Apr (MENA)--MENA has learned that detectives from the State Security Department arrested several terrorist extremists in al-Jizah's al-Munib district today. State Security investigators earlier obtained accurate information that several terrorists in the al-Munib area were planning terrorist acts. A special unit of the State Security forces stormed their hideout, arrested them, and thwarted their planned attacks.
