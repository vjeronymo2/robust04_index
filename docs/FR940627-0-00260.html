Measurement of protein using bicinchoninic acid. Annal. Biochem. 150:76&hyph;85. §79.68 Salmonella typhimurium reverse mutation assay. (a) Purpose. The Salmonella typhimurium histidine (his) reversion system is a microbial assay which measures his ^ c his + reversion induced by chemicals which cause base changes or frameshift mutations in the genome of the microorganism Salmonella typhimurium. (b) Definitions. For the purposes of this section, the following definitions apply: Base pair mutagen means an agent which causes a base change in DNA. In a reversion assay, this change may occur at the site of the original mutation or at a second site in the chromosome. Frameshift mutagen is an agent which causes the addition or deletion of single or multiple base pairs in the DNA molecule. Salmonella typhimurium reverse mutation assay detects mutation in a gene of a histidine-requiring strain to produce a histidine independent strain of this organism.
