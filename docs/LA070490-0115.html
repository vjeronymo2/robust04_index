HOW FIREWORKS GET THEIR COLORS

The fireworks' color palette is created by using precise amounts of chemicals mixed with the explosives. Blues, for example, are obtained by using copper, Strontium salts make the red colors, calcium makes orange and sodium makes yellow. Since World War II, organic compounds of chlorine have been mixed with the explosive to make the color burn brighter. The sparkling effects are created by using metals, such as iron or magnesium, which burn brightly when placed in an explosive mixture.

Source: Collier's Encyclopedia
