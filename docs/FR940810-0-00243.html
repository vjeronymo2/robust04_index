Introduction This method describes the collection of airborne asbestos fibers using calibrated sampling pumps with mixed-cellulose ester (MCE) filters and analysis by phase contrast microscopy (PCM). Some terms used are unique to this method and are defined below: Asbestos: A term for naturally occurring fibrous minerals. Asbestos includes chrysotile, crocidolite, amosite (cummingtonite-grunerite asbestos), tremolite asbestos, actinolite asbestos, anthophyllite asbestos, and any of these minerals that have been chemically treated and/or altered. The precise chemical formulation of each species will vary with the location from which it was mined.
