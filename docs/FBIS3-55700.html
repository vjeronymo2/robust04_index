Language: English Article Type:BFN [By ITAR-TASS correspondent Svetlana Ivanova] [Text] St. Petersburg January 11 TASS -- An Algerian Navy submarine, built here a few years ago, is now in a dock of the St. Petersburg enterprise "Korabelnye Verfi" for a major overhaul which is scheduled to be completed by the end of 1994. The enterprise has already received an order to restore another Algerian submarine, also built in Russia.
