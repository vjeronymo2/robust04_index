(1) Vehicles using fuels/fuel systems inherently low in refueling emissions are not required to conduct testing to demonstrate compliance with the refueling emission standards set forth in §§86.099&hyph;8, 86.001&hyph;9 or 86.004&hyph;9: Provided, that_ (i) This provision is only available for petroleum diesel fuel.
