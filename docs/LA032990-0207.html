March 29, 1990, Thursday, Orange County Edition

THE PENALTY PHASE

Nearly 16,000 Orange County individuals and businesses have been notified by the state Franchise Tax Board that they were required to, but did not, file a state tax return in 1988. The total assessments, as well as the dollar amount, for the five-county region were as follows:
