October 18, 1990, Thursday, Home Edition

4TH CHESS MATCH GAME ADJOURNED

The fourth game of the world championship chess match between champion Garry Kasparov and challenger Anatoly Karpov was adjourned Wednesday night after 40 moves.

Play was halted at 10:30 p.m. when Kasparov wrote down his 41st move and sealed it in an envelope.

The game is to be resumed this afternoon.

Kasparov, with 2 points to Karpov's 1, had the white pieces in the latest game of the $3-million world championship match. Game 3 ended in a draw Tuesday night. Kasparov won the second game on Oct. 10. The first game, played Oct. 8, ended in a draw.
