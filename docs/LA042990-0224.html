April 29, 1990, Sunday, Orange County Edition

CS FULLERTON SPLITS SOFTBALL DOUBLEHEADER

Cal State Fullerton gained a split with Cal Poly Pomona in a Big West Conference softball doubleheader Saturday at Pomona.

Fullerton (42-17, 20-9 in conference) won the first game, 2-0, on run-scoring singles by Missy Coombes and Charlotte Wiley. Pomona (27-30, 10-18) won the second game, 1-0, behind the one-hit pitching of Bonnie Ebenkamp.

In California Collegiate Athletic Assn. softball:

Cal State Bakersfield 2-1, Chapman 1-0 -- Trina Honea had Chapman's (36-17, 7-11) only run batted in at Bakersfield (33-11, 11-7).
