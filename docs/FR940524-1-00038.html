Section 84.206␣Particulate Tests; Respirators With Filters; Minimum Requirements; General This section would be redesignated from existing §11.162&hyph;7. It would be modified only to delete reference to various ``classifications'', such as paints and pneumoconiosis and fibrous producing dusts, that would no longer be appropriate with the introduction of the ``particulates'' classification with the new instantaneous-penetration test.
