920319 FT 19 MAR 92 / Survey of Software at Work (56): New educational package Multimedia applications can be distributed on CD-ROM, a form of storage which provides very high capacity. Pictured, right, is a sequence from Creepy Crawlies, a new educational multimedia package for children developed by the UK company, Media Design Interactive. Video sequences and narrative can be easily invoked by selecting icons on the screen, using a mouse pointer.
