Executive Order 12778 This rule has been reviewed under Executive Order 12778, Civil Justice Reform. This rule: (1) Preempts all State and local laws and regulations that are in conflict with this rule; (2) has no retroactive effect; and (3) does not require administrative proceedings before parties may file suit in court challenging this rule.
