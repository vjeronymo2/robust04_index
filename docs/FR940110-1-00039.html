EXAMINATIONS FEE ACCOUNT Current Fee and Proposed Fee Petition for a Nonimmigrant Worker (I&hyph;129) Current Fee Structure: The fee for a petition is a base fee of $70 plus either: _$10 per worker if requesting consulate or port-of-entry notification for visa issuance or admission; _$80 per worker if requesting a change of status; or _$50 per worker if requesting an extension of stay. Proposed Fee Structure: If filing for one worker, $120 per petition plus $10 for each dependent. If filing for more than one worker, $100 per petition plus $20 for each worker.
