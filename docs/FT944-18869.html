941001 FT 01 OCT 94 / Letters to the Editor: Pools expertise provides link Sir, With reference to the note 'A rash move' (Observer, September 27), the link between Peaudouce and Ladbroke's Vernons is clearly pools. A man as experienced as Philip Jarrold in mopping up operations is eminently qualified to manage clean sweeps, while avoiding soaking the rich.
