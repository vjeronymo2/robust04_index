January 30, 1990, Tuesday, Home Edition

FRESNO STATE ROUTS SAN JOSE STATE

Wilbert Hooker scored 16 points to lead Fresno State to a 69-53 Big West Conference victory against San Jose State Monday night at Fresno.

Tod Bernard added 15 points for Fresno State, which snapped a four-game losing streak and improved to 2-6 in conference and 7-10 overall.

San Jose State fell to 2-7 and 5-13.

Bernard had 14 of his points in the first half as Fresno State opened up a 36-25 lead. San Jose State got as close as seven points in the second half, but a 16-4 Fresno State run broke the game open.

Kenne Young led San Jose State with 14 points. Young's basket with 12:20 left in the game cut Fresno State's lead to 48-41, but the Spartans went almost five minutes without a basket before Mike Wasserburger hit a three-point shot.
