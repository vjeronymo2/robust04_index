920707 FT 07 JUL 92 / Survey of Software at Work (14): System Profile ALFA Asset Lease and Finance Administration is the name CHP has already given to the system it is developing for Hambros Leasing and which it hopes to sell as a package. The system handles finance leases and operating leases, back-to-back leases, hire purchase, off-shore leases and loans. Any of those financing methods can involve variable rates, repurchase agreements, multiple currencies, combined invoicing and maintenance, for example for contract hire. The treasury administration system caters for short-term loans, bills of exchange and swaps. All transactions, calculations and enquiries are dealt with on-line. Hambros Leasing has an IBM AS/400 serving 10 Idea colour terminals and eight personal computers. The value of the contract with CHP has not been disclosed.
