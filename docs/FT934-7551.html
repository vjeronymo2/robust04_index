931119 FT 19 NOV 93 / Mercury to link police forces MERCURY Communications, the main competitor to British Telecommunications, has won a contract to build a network to link Britain's 53 police forces. It will upgrade the existing police network and provide an integrated network for telephone, data and video communications. The Police National Network is the third large contract won by Mercury in the past two months. Last month it won a contract to run the government's long-distance inter-departmental network, which serves 250,000 civil servants in 550 offices.
