September 16, 1989, Saturday, Home Edition

'FINE-TUNING' THE MEDICARE SURTAX

So the Administration wants the catastrophic care legislation left untouched because the tax revenues generated thereby will help reduce the budget deficit. What this amounts to is a new tax on seniors only to pay for a program that only a small number will ever use in order to save the federal budget from breaching spending limits.

Read my lips on "No new taxes" -- ha!

JEANETTE YOUNG

Los Olivos
