March 19, 1990, Monday, P.M. Final

SHORT TAKES;

'BIGFOOT' HARRY GETS 3-YEAR RUN

"Harry and the Hendersons," a new Fox series produced by Universal Television and Amblin Entertainment, has been given an unprecedented 72-episode commitment.

The half-hour weekly sitcom, which will debut in January, is guaranteed a three-year run on the Fox network -- the longest commitment ever made to a syndicated or network show, according to the show's distributor, MCA.

"Harry," is based on the 1988 feature film about a Bigfoot that is hidden by a Seattle family. It will begin production this spring, with casting announced shortly.
