CSO [Article by Yu Yikuan [0151 5030 1401]: ``Fuzhou (Hangzhou)-Guilin-Chengdu Fiber Optic Cable To Be Built''] [Summary] It has been learned from authoritative sources that construction on China's longest fiber optic cable communications trunk line -- the Fuzhou (Hangzhou)-Guilin-Chengdu cable -- will begin this year. This 4,354-km-long, 1.071-billion-yuan trunk line will link up six provinces: Fujian, Zhejiang, Jiangxi, Hunan, Guizhou, and Sichuan.
