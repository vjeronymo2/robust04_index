930205 FT 05 FEB 93 / Technology (Worth Watching): Paradox database now on Windows Relational database software specialist Borland International has announced a version of its popular Paradox database management software for the Windows environment. Companies buying the package before April 30 will get it for the introductory price of Pounds 99.95. Paradox for Windows can store and handle scanned images, such as signatures, graphics, photos and even moving video images. Borland International: US, 408 439 1064; UK, 0734 320022.
