941028 FT 28 OCT 94 / Observer: Weighty matters Read the small print - always. The latest brochure - printed in English - from Matif, the French financial futures market, concerns the new rapeseed market, which opens in Paris today. Unfortunately, it translates world production of 'metric tonnes' into 'tons', without realising that they are not quite the same thing and that a conversion would be illuminating. Still, what's a few pounds - is that weight or currency? - between friends?
