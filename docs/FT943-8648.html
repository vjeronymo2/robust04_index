940816 FT 16 AUG 94 / UK Company News in Brief: Automated Security Holdings AUTOMATED Security Holdings' Californian subsidiary, API Security, has agreed to sell its residential alarm base for Dollars 2.3m (Pounds 1.48m). In addition, Sonitrol Management Corporation, a subsidiary of ASH, is to purchase the client base of Sonitrol Eugene (Oregon) for about Dollars 2.7m.
