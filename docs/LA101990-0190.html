October 19, 1990, Friday, P.M. Final

THE SIDELINES;

RECORD LAP AT JAPAN GRAND PRIX

Austrian Gerhard Berger set a record lap time today in earning the provisional pole position for the Japan Grand Prix.

With the one-hour time trial nearly over, Berger, driving a McLaren-Honda, circled Suzuka's 3.64-mile course in one minute, 38.374 seconds.

The time beat the previous mark of 1:39.493 set in last year's race by teammate Ayrton Senna.
