930924 FT 24 SEP 93 / Technology (Worth Watching): Escaping into 3-D Screen addicts can step into the third dimension with the aid of what Logitech of Switzerland calls the world's first interactive three-dimensional controller for computer games. Cyberman has been developed for the latest generation of PC-based games. It provides single-handed control of movements in all axes without the need to use the keyboard and has a tactile response mechanism so users can feel what is happening in the game. Logi (UK): UK, 0344 891313.
