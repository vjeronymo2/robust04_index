September 8, 1990, Saturday, Home Edition

BRIEFLY

Lotus to Upgrade 1-2-3 Software: On Monday, Lotus Development Corp. will start selling its Release 3.1 version of the best-selling Lotus 1-2-3 spreadsheet software for personal computers. Many analysts say the latest version should have been introduced 15 months ago instead of Release 3.0, a version that many customers rejected, contributing to Lotus' earnings woes.
