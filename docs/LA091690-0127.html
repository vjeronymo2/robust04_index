September 16, 1990, Sunday, Valley Edition

HIGH SCHOOL ROUNDUP;

CROSS-COUNTRY

Rotary West Ojai Invitational -- Fillmore's Jorge Barajas set a boys' course record with a time of 14 minutes 50 seconds in a 2.7-mile race at Thacher School. Barajas' time bettered the previous record of 14:53, set last year by Ventura's Roman Morales.

Fillmore's Maribella Aparicio set a girls' record with a time of 17:15, eclipsing the 19:03 set by Morro Bay's Jenny Lacovara last year.
