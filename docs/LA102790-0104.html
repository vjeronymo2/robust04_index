October 27, 1990, Saturday, Valley Edition

ROUNDUP;

NONLEAGUE

St. Francis 7, Chaminade 0 -- Senior fullback Larry Duenas scored on a one-yard run with 5:18 left in the game to lift St. Francis at home. The run capped a seven-play, 40-yard drive that was set up by an interception by defensive back Jesse Hall, who returned the ball 20 yards.

Duenas led St. Francis (5-3) with 65 yards in 14 carries. St. Francis gained only 126 yards to 192 for Chaminade (1-6-1). BRYAN RODGERS
