Koch will install the delivery tap for an estimated cost of $4,900 under its blanket certificate issued in Docket No. CP82&hyph;430&hyph;000, pursuant to Section 7(c) of the Natural Gas Act, all as more fully set forth in the request which is on file with the Commission and open to public inspection.
