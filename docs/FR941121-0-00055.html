For the reasons set out in the preamble, Title 30, Chapter VII, Subchapter T of the Code of Federal Regulations is amended as set forth below: PART 913_ILLINOIS 1. The authority citation for Part 913 is revised to read as follows: Authority: 30 U.S.C. 1201 et seq. 2.
