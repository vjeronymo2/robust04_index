BFN [Unattributed report: "A Package of Amendments to Banking Legislation Worked out in State Duma"] [Text] The Russian State Duma's banking legislation subcommittee has drafted a package of amendments and addenda to the Russian Federation Law "On Banks and Banking Activity." The changes would make it possible to correct the deficiencies "preventing the normal functioning of the banking system." This has been advised by Pavel Medvedev, chairman of the Duma banking legislation subcommittee. According to him, the subcommittee is hoping to submit a corresponding document to be considered by the State Duma in the first reading till the end of April.
