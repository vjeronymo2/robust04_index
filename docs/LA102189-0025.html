October 21, 1989, Saturday, Home Edition

CS FULLERTON FACES UNBEATEN FRESNO STATE

Cal State Fullerton will take on Fresno State and the Bulldogs' 13-game winning streak in a Big West Conference game at 7 tonight in Fresno's Bulldog Stadium.

Fresno State (6-0 overall, 3-0 in the conference) began its streak with a victory over Fullerton last Oct. 8 and would set a school record for consecutive victories if it wins tonight. The Bulldogs' streak is the second longest in the nation in Division I-A to Notre Dame's 18.

Fresno's offense, led by running backs Aaron Craver and Myron Jones, is averaging 42 points a game.

Fullerton (3-3-1, 2-1) hasn't beaten Fresno State since 1984.

Fullerton running back Mike Pringle, who leads the nation in all-purpose running, needs 133 yards to become only the fourth Titan to rush for 1,000 yards in a season. ROBYN NORWOOD
