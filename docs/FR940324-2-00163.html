SF&hyph;424A_``Budget Information_Non-Construction Programs'' See instructions accompanying this form as well as the instructions set forth below: In completing these sections, the ``Federal Funds'' budget entries will relate to the requested OCS discretionary funds only, and ``Non-Federal'' will include mobilized funds from all other sources_applicant, state, local, and other. Sections A and D of SF&hyph;424A must contain entries for both Federal (OCS) and Non-Federal (mobilized) funds. Section B contains entries for Federal (OCS) funds only.
