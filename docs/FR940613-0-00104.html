The length of each trailing unit is limited to 28.5 feet. This describes a two trailing unit vehicle whose operation is guaranteed by the STAA of 1982 regardless of inter-unit spacing. As long as each trailing unit is 28.5 feet long or less, cargo-carrying length is not restricted. This combination is listed as a LCV because it can exceed the 80,000-pound threshold established in the Congressional definition. The 86,400-pound gross weight limit is grandfathered for New Mexico.
