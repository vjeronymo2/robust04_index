A new section 572.705 is added, as follows: §572.705 Monitoring report for Class C agreements. This section sets forth the Monitoring Report form for Class C agreements, with accompanying instructions that are intended to facilitate the completion of the Report. The explanation and instructions should be read in conjunction with the Shipping Act of 1984 and this part 572. Monitoring Report for Class C Agreements Instructions A complete response must be supplied to the Report. Where the filing party is unable to supply a complete response, that party shall provide either estimated data (with an explanation of why precise data are not available) or a detailed statement of reasons for noncompliance and the efforts made to obtain the required information. All sources must be identified.
