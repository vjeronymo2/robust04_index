BFN [Text] Aswan, 20 Apr (MENA)--A responsible security source in the Aswan Security Directorate has denied a report published recently by an opposition paper claiming that a group calling itself Jama'at al-Intiqam [The Revenge Group] in the Aswan Governorate has raised 250,000 pounds from the governorate's inhabitants to finance retaliatory operations against the police agencies in revenge for the death of eight members of the group in Markaz Idfu. The security source stressed that the security organs in the governorate are on full alert to confront any terrorist actions and abort them immediately. It added that there are no moves to raise funds in the Aswan Governorate for this or any other purpose. It stressed that security in the governorate is stable following the recent security blows dealt by the security organs to the extremist terrorist elements.
